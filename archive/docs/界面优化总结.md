### 2. 🔍 智能股票输入系统

**新增模块**: `src/ui/smart_stock_input.py`

**核心功能**:
- 📊 **智能选择**: 115只股票下拉选择
- 🔍 **实时搜索**: 支持代码和名称模糊搜索
- ✅ **自动匹配**: 输入代码自动显示对应名称
- 💡 **智能建议**: 提供相似股票建议

**搜索能力**:
```python
# 代码搜索示例
"000001" → 000001.SZ - 平安银行
"600519" → 600519.SH - 贵州茅台
"AAPL" → AAPL - 苹果公司

# 名称搜索示例
"平安" → 000001.SZ - 平安银行
"茅台" → 600519.SH - 贵州茅台
"苹果" → AAPL - 苹果公司
```
